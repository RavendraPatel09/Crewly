## 🎯 Project Overview

This platform enables:
- Creators to build profiles and post collaboration requests
- Service providers to showcase skills and portfolios
- Users to discover talent across multiple social platforms
- Chat experience (UI only)
- AI-powered suggestions (mocked frontend features)

---

## ✨ Features

### 👤 User System (Frontend Only)
- Creator & Service Provider roles
- Login / Signup UI (no backend logic)

### 🧑‍💼 Profiles
- Creator profiles with social links (YouTube, Instagram, TikTok, etc.)
- Provider profiles with skills, pricing, and portfolio
- Editable UI state (mocked)

### 📢 Collaboration Feed
- Social media-style feed
- Create collaboration posts
- Like/comment UI (mock only)

### 🔍 Explore System
- Search creators/providers
- Filters:
  - Platform (YouTube, Instagram, TikTok, Facebook)
  - Skills
  - Category
- Card-based results UI

### 💬 Chat UI (Mock Only)
- Messenger-style chat interface
- Dummy conversations
- Send message UI (no backend)

---

## 🤖 AI Features (Mocked UI Only)

- AI Bio Generator
- AI Collaboration Suggestion Engine
- AI Caption / Script Generator
- Smart Match Recommendations

⚠️ No API calls used — only UI simulation
